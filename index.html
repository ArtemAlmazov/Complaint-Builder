    О проекте

    Complaint Builder — удобный генератор форумных жалоб для RP-проектов. Он помогает игрокам и администрации быстро оформлять темы в BBCode, проверять обязательные поля и копировать готовый результат для публикации на форуме.

    • Понятный интерфейс с акцентом на скорость.
    • Проверка обязательных условий перед отправкой.
    • Полностью статический сайт для GitHub Pages.
    Для SAMP / CRMP / MTA / RP проектовby A. Almazov
