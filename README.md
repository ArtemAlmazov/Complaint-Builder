# Complaint Builder

Complaint Builder — статический сайт для RP-проектов, который помогает быстро подготовить форумную жалобу в формате BBCode.

## Что реализовано (v1.0.1)

- Современный тёмный интерфейс (dark premium UI)
- Full-width layout: шапка, hero, генератор, преимущества и footer на всю ширину
- Разделы: Главная, Создать жалобу, Создать заявление, Ответ администрации, О проекте
- Полностью рабочий модуль **«Создать жалобу»**:
  - проверка обязательных полей
  - предупреждения по /c 060
  - напоминание про тайм-код для длинного видео
  - генерация BBCode
  - копирование BBCode в буфер обмена
  - очистка формы

## Запуск локально

Откройте `index.html` в браузере.

## Публикация через GitHub Pages

1. Загрузите проект в GitHub-репозиторий.
2. Откройте `Settings` → `Pages`.
3. В разделе **Build and deployment** выберите:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` (или ваша ветка) и папку `/ (root)`
4. Нажмите **Save**.
5. Дождитесь публикации — GitHub выдаст ссылку вида:
   `https://<username>.github.io/<repository>/`

Сайт полностью статический и не требует Node.js или backend.
