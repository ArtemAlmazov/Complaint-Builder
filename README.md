# Complaint Builder

Complaint Builder — статический сайт-конструктор форумных тем для RP-проектов.

## Возможности v2.0
- Генерация жалоб в BBCode
- Шаблоны проектов: Универсальный / CRASH CRMP / RADMIR RP / Свой шаблон
- Свой шаблон с сохранением в localStorage
- Проверка риска отказа
- Улучшение описания через локальные шаблоны
- Модуль «Ответ администрации»
- Модуль «Заявление»
- Модуль «Черновики» (localStorage)
- Копирование BBCode и скачивание `.txt`
- Работа без backend, готово для GitHub Pages

## Запуск
- Локально: откройте `index.html` в браузере.
- GitHub Pages: загрузите репозиторий и включите Pages для ветки `main` и папки `/ (root)`.
